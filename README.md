# Noitreign

Time-pressure mod for [Noita](https://noitagame.com/) inspired by Elden Ring: Nightreign.

A countdown timer per biome forces you forward -- when it expires, escalating curse damage punishes lingering. In exchange, each Holy Mountain grants bonus HP and a random perk.

## Installation

Copy this folder into your Noita mods directory, then enable **Noitreign** from the in-game mods menu.
